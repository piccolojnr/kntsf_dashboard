## How to compile

- **Option A**: Open each file in order in your editor and copy-paste into a single Word/Google Doc, then apply formatting (headings, page breaks, TABLE OF CONTENTS in bold, etc.) per KUC guideline.  
- **Option B**: Use a tool (e.g. Pandoc, md-to-pdf) to merge the .md files in the order above and export to PDF; then adjust formatting as needed.  
- **Option C**: Use the list above as the source for your actual Table of Contents; generate page numbers after final layout.

Ensure front matter uses the exact headings and layout required by the guideline (e.g. title page five blocks, declaration placeholders for signatures, abstract ≤250 words, no symbols/diagrams in abstract).
