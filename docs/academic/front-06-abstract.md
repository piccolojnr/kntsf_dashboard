This project presents the Knutsford University SRC Dashboard, a web-based administrative system for managing student permits, campus events, news, newsletters, documents, polls, and payments. The system addresses manual permit processing, scattered student services, and the lack of real-time verification at the university. Built with Next.js, TypeScript, Prisma, and MySQL, it provides role-based access for administrators, SRC executives, and staff; integrates Paystack for permit fees and Cloudinary for media; and supports permit verification via unique QR codes. The report covers the background and problem statement, objectives, literature and review of related systems, system analysis and design including database and UI design, implementation with selected code evidence, testing, and deployment. The main conclusions are that the system reduces permit processing time, centralises student-life management, and provides a scalable foundation for future enhancements such as mobile applications and ERP integration. *(Edit and shorten as needed to stay within 250 words.)*
